crear una aplicacion
    vue create journal

Para instalar bootsrap
    yarn add bootstrap

Para usar sass debemos instalar sass
    yarn add -D sass-loader@10 sass

Debemos de configurar nuestro proyecto para que el webpack use sass
Creamos el siguiente archivo
    styles/styles.scss y dentro importamos bootsrap.scss con la siguiente linea
    @import 'node_modules/bootstrap/scss/bootstrap.scss'
Despues nos vemos al main.js y configuramos este para usar sass el archivo styles.scss
    import "./styles/styles.scss";
